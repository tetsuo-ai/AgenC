# Private Task Demo

This directory is a transitional rollback mirror of the curated SDK example that now lives in [`tetsuo-ai/agenc-sdk`](https://github.com/tetsuo-ai/agenc-sdk/tree/main/examples/private-task-demo).

The monorepo no longer owns this example as a workspace. Prefer the standalone repo copy. If you need to run the local mirror directly:

```bash
npm install --no-fund --no-package-lock --prefix examples/private-task-demo
npm run start --prefix examples/private-task-demo -- --help
```
